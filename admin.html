Welcome, Admin

Manage all site content and deploy directly to GitHub.

💡 Activate from the website: Open your live site and type CARYO (all caps) anywhere on the page — the admin panel opens automatically in a new tab.
✅ Auto-sync: Every 💾 Save button instantly pushes your changes to site-data.json in your GitHub repo — no extra steps needed. The website reads this file on every load.
🚀 GitHub Deploy: Use the Deploy button to also bake content into index.html itself (for maximum compatibility / CDN caching).
⚠️ Image uploads: Images are converted to base64 references or you provide the path. For real image hosting, commit them to your GitHub repo's assets/ folder.
