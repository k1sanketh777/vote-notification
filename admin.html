Welcome, Admin

Manage all site content and deploy directly to GitHub.

💡 Activate from the website: Open your live site and type CARYO (all caps) anywhere on the page — the admin panel opens automatically in a new tab.
☁️ How saving works: Every 💾 Save button writes your changes directly to site-data.json in your GitHub repo as a new commit. Your live website fetches this file on every load — no extra deploy step needed.
🔄 Always in sync: When you open the admin panel, it automatically loads the latest site-data.json from GitHub so you always see the current live data.
⚠️ You opened admin.html as a local file (file://). GitHub API is blocked — saves will auto-download a JSON file instead. Upload that file to your repo on github.com to apply changes.
⚠️ Images: Add image files directly to your GitHub repo under assets/images/, then enter the path in the admin panel (e.g. assets/images/Team/photo.png).
